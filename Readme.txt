Instructions:

open terminal in this folder and run the command "make" to create the object files and executables

run the program "./simulator &" to run the simulator in the background
run the program "./customer num_from_0_to_6 num_from_0_to_6" to simulate a taxi request
run the program "./generator &" to simulate a bunch of customer requests
run the program "./stop" to shutdown the dispatch server

run command "make clean" to clear all files
